# Flowbase Cozy UI Theme

## Design Direction
Flowbase should feel calm, capable, and fresh: a bright workspace for planning, writing, mapping ideas, and getting help from AI without visual clutter. The product UI opens directly into useful work surfaces rather than marketing sections.

## Color Palette
- Background: luminous warm off-white `hsl(42 82% 97%)` for a clean app canvas.
- Surface: crisp white `hsl(0 0% 100%)` for cards, panels, and sidebars.
- Text: deep ink `hsl(222 28% 14%)` for sharper readability.
- Primary: fresh coral `hsl(8 82% 62%)` for selected states and primary actions.
- Accent families:
  - Coral and rose for creative or active states.
  - Mint, sage, and teal for spaces, collaboration, and calendar moments.
  - Amber for kanban, templates, and optimistic highlights.
  - Violet and sky for AI features and lightweight system affordances.
- Borders: warm linen `hsl(36 45% 86%)`, used softly and consistently.

## Typography
- Use the system sans stack with Inter first when available.
- Keep headings crisp and readable, with normal letter spacing.
- Dashboard headings should be confident but not oversized; compact panels use small, scannable titles.
- Body copy uses relaxed line-height for notes and concise metadata text for dense views.

## Spacing And Shape
- Base rhythm: 8px.
- Preferred radius: 8px for buttons, cards, sidebar controls, and whiteboard objects.
- Use generous page gutters on desktop and tighter but stable padding on mobile.
- Fixed-format UI elements, such as sidebar controls and icons, should have stable dimensions to avoid layout shifts.

## Interaction Guidelines
- Icons should be Lucide-based and colorful by feature area.
- Hover states should use soft warm fills, not high-contrast flashes.
- Focus states should remain visible through the primary coral ring color.
- Collapsed navigation keeps icons centered and accessible via labels or titles.
- Avoid decorative gradient blobs, nested cards, and landing-page hero layouts inside the app shell.
